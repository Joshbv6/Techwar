Tech-War

Video Demo URL: https://youtu.be/557QaN68_TY

Description: 
Tech-War is the idea of a future browser game. I'm still working of it, but for now, this is what I've got.
It is a register form. The app makes sure that you write a valid password and inserts your info into a database called 'techwar.db'.
password is inserted as a hash, for security, and as soon as your info gets inserted into the database, the app itself sends you an email.
